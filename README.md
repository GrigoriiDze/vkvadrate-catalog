# vkvadrate — Каталог товаров

React-приложение с каталогом, фильтрами и кнопкой WhatsApp.

---

## ⚡ Быстрый старт

```bash
npm install
npm run dev
```

Открой http://localhost:5173

---

## 📁 Структура файлов

```
vkvadrate-catalog/
├── public/
│   ├── logo.png          ← Логотип (скопируй свой)
│   └── images/
│       ├── K01.jpeg      ← Все фото сюда
│       ├── K02.jpeg
│       └── ...
├── src/
│   ├── App.jsx           ← Весь код каталога
│   └── index.css         ← Стили
├── index.html
└── package.json
```

---

## 📸 Добавить фотографии

Скопируй все JPEG-файлы товаров в папку `public/images/`

Имена файлов должны совпадать с артикулами из CSV: `K01.jpeg`, `K02.jpeg`, `K043.jpeg` и т.д.

---

## 📱 Изменить номер WhatsApp

Открой `src/App.jsx`, найди строку в самом начале:

```js
const WHATSAPP = '77XXXXXXXXX'
```

Замени на свой номер **без + и пробелов**, например:
```js
const WHATSAPP = '77001234567'
```

## 🚀 Деплой на хостинг

### Vercel (рекомендую, бесплатно)
1. Зарегистрируйся на vercel.com
2. Загрузи папку проекта или подключи GitHub
3. Vercel сам запустит `npm run build`

### Netlify
1. Собери: `npm run build`
2. Загрузи папку `dist/` на netlify.com/drop

### Любой хостинг
```bash
npm run build   # создаст папку dist/
```
Загрузи содержимое `dist/` на хостинг.

---

## ➕ Добавить новый товар

В `src/App.jsx` найди массив `PRODUCTS` и добавь строку:

```js
{ id: 'K100', name: 'Название товара', category: 'Скамьи', price: 350000 },
```

Категории: `Скамьи`, `Перголы`, `МАФ`, `Освещение`, `Спортивное оборудование`

---

## 🎨 Изменить цвета

В `src/index.css` в самом начале — CSS-переменные:

```css
--bronze: #C4894A;   /* акцентный цвет */
--navy:   #0C1B33;   /* тёмный цвет */
```
